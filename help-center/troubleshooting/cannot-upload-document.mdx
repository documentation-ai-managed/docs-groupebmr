---
title: Cannot Upload a Document
description: Resolve document upload errors in Groupebmr.
---

## Symptoms

- Upload fails or hangs
- Error: "Upload failed, try again"

## Quick Fix

<Steps>
  <Step title="Check your file size">
    Groupebmr accepts files up to 20MB. Reduce file size if needed.
  </Step>
  <Step title="Verify file type">
    Allowed: PDF, DOCX, PNG, JPG. Convert if unsupported.
  </Step>
  <Step title="Refresh and retry">
    Reload the page, then upload again.
  </Step>
</Steps>
