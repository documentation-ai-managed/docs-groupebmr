## Webhook Configurations

Set up webhooks to receive real-time events from Groupebmr, such as project updates or user actions.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to Dashboard > Settings > Webhooks > New Webhook.
    
    Enter your endpoint URL, like `https://your-webhook-url.com/groupebmr`.
  </Step>
  <Step title="Select Events" icon="list">
    Choose events: `project.updated`, `user.created`.
  </Step>
  <Step title="Verify and Test" icon="check-circle">
    Test the webhook and verify the signature using the shared secret.
  </Step>
</Steps>

Handle incoming payloads securely. Verify the signature to prevent tampering.

<CodeGroup tabs="Node.js,Python">
  ```javascript
  const crypto = require('crypto');

  function verifySignature(payload, signature, secret) {
    const hmac = crypto.createHmac('sha256', secret);
    const digest = 'sha256=' + hmac.update(payload).digest('hex');
    return crypto.timingSafeEqual(
      Buffer.from(signature),
      Buffer.from(digest)
    );
  }
  ```
  ```python
  import hmac
  import hashlib

  def verify_signature(payload, signature, secret):
      expected = 'sha256=' + hmac.new(
          secret.encode(), payload.encode(), hashlib.sha256
      ).hexdigest()
      return hmac.compare_digest(signature, expected)
  ```
</CodeGroup>

## Custom Integrations via API

For advanced use cases, integrate directly with the Groupebmr API at `https://api.example.com/v1`.

<Request tabs="cURL,JavaScript">
  ```
  curl -X POST https://api.example.com/v1/integrations \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{"type": "custom", "config": {"url": "https://your-service.com"}}'
  ```
  ```javascript
  const response = await fetch('https://api.example.com/v1/integrations', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      type: 'custom',
      config: { url: 'https://your-service.com' }
    })
  });
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "id": "int_456",
    "status": "active",
    "config": {
      "url": "https://your-service.com"
    }
  }
  ```
</Response>

<ParamField path="config.url" param-type="string" required="true">
  Your custom endpoint URL.
</ParamField>

<Expandable title="Advanced API Security" default-open="false">
  Rotate API keys regularly and use IP restrictions. Monitor usage via dashboard logs.
</Expandable>
