## Overview

Groupebmr provides a centralized space for your team's documentation needs. You can create, organize, and collaborate on project docs with ease. Use it to maintain living documentation that stays up-to-date as your projects evolve.

## Key Benefits for Teams

Groupebmr streamlines documentation workflows. Teams benefit from real-time collaboration, version history, and powerful search across all documents. Scale your knowledge base without complexity.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your free account at `https://groupebmr.com/signup`.
  </Step>
  <Step title="Create Workspace" icon="home">
    Set up a new workspace for your project.
  </Step>
  <Step title="Add First Doc" icon="plus">
    <CodeGroup tabs="Markdown,API">
      ````markdown
      # My First Document
      
      Welcome to Groupebmr!
      
      ## Quick Tips
      - Use headings for structure
      - Add code blocks for examples
      ````
      ````javascript
      // Example API call
      const response = await fetch('https://api.groupebmr.com/docs', {
        headers: { 'Authorization': `Bearer YOUR_API_KEY` }
      });
      ````
    </CodeGroup>
  </Step>
  <Step title="Invite Team" icon="mail">
    Share access with your collaborators.
  </Step>
</Steps>
